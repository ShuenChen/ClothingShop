To open the main page run the index.php file under the main Clothing folder.

The admin panel will be situated under the Admin folder however it can only
be accessed from the main login page under login.php in the Clothing main. There will be an
option to log in as an admin under the login page.

The steps to the admin panbel will look like this:

/Clothing/index.php ---> /Clothing/login.php  ---> /Clothing/Admin/Admin_Login_Page.php
